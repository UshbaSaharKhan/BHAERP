# BHA ERP

## Overview
BHA ERP is a client comprehensive Enterprise Resource Planning (ERP) system designed to streamline and automate core business operations.
The platform integrates multiple business functions into a single system, improving operational efficiency, data accuracy, and decision-making.

## Features

- Financial Accounting
- Accounts Payable & Accounts Receivable
- Inventory & Stock Management
- Procurement & Purchasing
- Sales & Order Management
- Warehouse Management
-  Human Resource Management (HR)
-  Payroll Management
-  Reporting & Analytics
- Core ERP Administration
-  Role-Based Access Control

## Technology Stack

-  ASP.NET MVC & ASP.NET Core
- Entity Framework Core (EF Core)
- SQL Server
- jQuery
- HTML5
- CSS3
- Bootstrap
- JavaScript
- RESTful APIs

## Purpose

The system is designed to help organizations manage their financial, operational, and administrative processes through a centralized ERP platform, 
enabling improved productivity and business performance.

> Note: This repository contains a private ERP project intended for development and version control purposes.
